# Contributing

Thanks for your interest in this project! Please read this before opening an issue or
trying to contribute code.

## Current contribution policy

**Pull requests are currently disabled on this repository.** This project is
solo-maintained, and I implement changes myself. This isn't personal — it's just how the
project runs right now.

If this changes in the future, this file will be updated accordingly.

## Reporting bugs or suggesting features

Issues are still welcome and read. When opening one, please include:

- A clear description of the problem or suggestion
- Steps to reproduce (for bugs)
- Your environment (OS, version, relevant config)

## About labeled issues

Some issues in this repo carry labels (e.g. `sonnet`) that reflect my own internal
workflow — these indicate an issue is scoped for me to implement myself, often with AI
assistance. **These labels are not an invitation for outside contributors to submit a PR
against them.** Since PRs are disabled, this is moot for now, but it's worth stating
plainly for anyone browsing the issue tracker.

## Before assuming an issue is "easy"

Please don't assume an issue is trivial just because it reads as short or
self-contained. This project has setup steps (e.g. generated files, build steps) that
aren't always obvious from the issue text alone. If you're unsure whether something is
safe to act on, ask first — don't guess.

## Questions or discussion

Feel free to open an issue for questions or discussion. I read everything, even if I
can't always respond quickly.

## Code of conduct

Be respectful. Don't spam issues. Don't submit AI-generated content you haven't
personally verified against a working build of the project.
